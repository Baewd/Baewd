-- От <body> до </body> росположено видимое тело кода
--Закрывать теги нужно сначало 1который в нутрии а потом который с наружи :
<head>…. <b>…. </b>…. <head>

                   < > - тег
--Тег – функция для описания структуры и вида текста на сайте
 
                   Виды :
---<h>..….</h>- тег для заголовков бывает от h1 до h6( где h1 самый большой  а h6 самый маленький)
---<p>..….</p>- тег для создания абзацев 
--..............<br>- тег ставится в конце для создание новой строки после ставится 1
--..............<hr>- создаёт горизонтальную линию на строке ниже после себя
---<b>…...</b>- тег для создания жирного текста
-<em>....</em>- тег для создания курсивного текста
-<strike>..</strike>- тег для создания зачёркнутого текста
-<sub>..</sub>- тег для создание символа ниже обычного
-<sup>..</sup>- тег для создание символа вышеобычного
-<abr>...</abr>- тег в котором записываются абривиатуры
--<address>...</address> - тег в котором пишется адрес
--<small>...</small>- делает элемент чуть меньше чем всё остальное
-<big>...</big>- делает элемент чуть больше чем всё остальное
-<pre>...</pre>- сохраняет все отступы при выводе текста
-<code>...</code>- в нём пишетя код
--<ol>.....</ol>- используется для создания пронумеровонного списка
--<li>.......</li>- используется c <ol> и с <ul> для прописывания элементов в списке
--<ul>.....</ul>- используется для создания непронумеровонного списка
